# Retro Game Vault

Retro Game Vault არის ბრაუზერზე დაფუძნებული ვებ აპლიკაცია, სადაც მომხმარებლებს შეუძლიათ მოძებნონ და აღმოაჩინონ უფასო რეტრო და ბრაუზერული თამაშები, შეინახონ საყვარელი თამაშები პერსონალურ ბიბლიოთეკაში, და ნახონ თითოეული თამაშის დეტალური ინფორმაცია მოდალურ ფანჯარაში — ყოველივე ეს სრულიად უფასოდ, რეგისტრაციის გარეშე, მხოლოდ სახელის შეყვანით.

## ტექნოლოგიური სტეკი

| ტექნოლოგია | გამოყენება |
|---|---|
| HTML5 | სტრუქტურა და სემანტიკა |
| CSS3 | სტილი და განლაგება |
| Vanilla JavaScript (ES Modules) | ლოგიკა, DOM მანიპულაცია |
| [Free-to-Play Games API](https://www.freetogame.com/api) | თამაშების მონაცემები |
| `localStorage` | ფავორიტების შენახვა, სესიის მართვა |
| Cookies | ავტორიზაციის სტატუსი |

> არანაირი ფრეიმვორკი, არანაირი ბექენდი, არანაირი build step.

```
final-template/

│   ├── login.js        ← შესვლის ლოგიკა (cookie + redirect)
│   ├── main.js         ← entry point index.html-ისთვის
│   └── saved.js        ← entry point saved.html-ისთვის
├── assets/             ← სტატიკური ასეტები (სურათები და სხვ.)

```

## როგორ გავუშვა

1. გადმოიწერე ან clone გააკეთე რეპოზიტორია.
2. გახსენი **`index.html`** ბრაუზერში (ორმაგი კლიკი საკმარისია — სერვერი **არ არის საჭირო**).
3. ავტომატურად **`login.html`**-ზე მოხვდები, რადგან ავტორიზაცია სავალდებულოა.
4. შეიყვანე შენი სახელი და დააჭირე "შესვლა" — გადამისამართდები მთავარ გვერდზე.
5. მოძებნე თამაშები, გახსენი დეტალები, შეინახე ფავორიტები.

> **შენიშვნა:** ES Modules (`type="module"`) Chrome/Firefox/Edge-ში `file://` პროტოკოლით სწორად მუშაობს. Safari-ზე CORS შეზღუდვის გამო შეიძლება საჭირო გახდეს მარტივი ლოკალური სერვერი (მაგ. `npx serve .` ან VS Code Live Server).

## ფუნქციონალი

- 🔍 **ძიება** — თამაშების ფილტრაცია სახელით, ჟანრით და პლატფორმით
- ❤️ **ფავორიტები** — თამაშების შენახვა / წაშლა `localStorage`-ში
- 🪟 **მოდალი** — დეტალური ინფორმაცია (სურათი, აღწერა, ჟანრი, პლატფორმა) popup-ში
- 👤 **სესია** — სახელზე დაფუძნებული "შესვლა / გასვლა" cookie-ს საშუალებით
